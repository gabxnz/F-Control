# F-Control

**Problema Real:** A dificuldade de organização financeira e o endividamento por falta de controle de gastos simples.
**Solução:** Uma ferramenta intuitiva para registro de entradas e saídas, permitindo visualização clara do saldo.
**Público-alvo:** Microempreendedores e pessoas que buscam saúde financeira.

## 🚀 Tecnologias
- HTML5 / CSS3 / JavaScript
- Node.js & npm
- Jest (Testes)
- ESLint (Linting)
- GitHub Actions (CI)

## 🛠️ Como Executar
1. Instale as dependências: `npm install`
2. Rode a aplicação: Abra o `index.html` no navegador.
3. Rode os testes: `npm test`
4. Rode o lint: `npm run lint`
